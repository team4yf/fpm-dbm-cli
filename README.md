# FPM-DBM-CLI

A tool for generate or execute meta SQL.

## Install

```bash
$ npm i -g fpm-dbm-cli
```

## Usage

```bash
$ fpm-dbm -f ./meta/*.json -d ./sql -o
```

## Argument Defined

- `-f`: set the directory contains the meta.json files
- `-d`: set the SQL file output path
- `-o`: show the SQL on the console
